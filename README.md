# 👥 Team Task Manager

## 📌 Description

**Team Task Manager** is a web-based application that helps teams efficiently manage and track their tasks.
It allows users to create, assign, update, and monitor tasks in a collaborative environment.



## 🚀 Features

*  User Authentication (Login/Signup)
*  Task Creation & Assignment
*  Task Status Tracking (Pending, In Progress, Completed)
*  Team Collaboration
*  Dashboard for Task Overview


##  Tech Stack

* **Backend:** Java / Spring Boot
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** MySQL
* **Version Control:** Git & GitHub
